# Clove.geolocation.clearWatch(wId)

Dismantles an active background location monitoring stream completely, freeing hardware resources and unbinding callback listener dependencies.

## Parameters

| Parameter | Type | Description | Example |
| :--- | :--- | :--- | :--- |
| `wId` | `number` | The specific tracking registration identifier key token previously generated by `Clove.geolocation.watchPosition`. | `3` |

## Return Type

* **Type:** `Promise<void>`
* **Description:** Resolves with no value once the physical hardware positioning listener instance is shut down natively.

## Code Example

```javascript
async function haltActiveTelemetryPipelines() {
  // Assuming globalTrackingToken holds an active ID integer resolved from watchPosition
  if (globalTrackingToken === null) {
    console.warn("No active spatial tracking engines currently flagged for shutdown operations.");
    return;
  }

  try {
    await Clove.ready();
    console.log(`Severing active coordinate streaming handlers linked to token ID: ${globalTrackingToken}`);
    
    await Clove.geolocation.clearWatch(globalTrackingToken);
    console.log("Native hardware event listener dismantled securely. Streaming halted.");
    
    globalTrackingToken = null; // Clean context references
    
  } catch (error) {
    console.error("Failed to cleanly disconnect the active geolocation tracking loop sequence:", error);
  }
}
```
